# Простой сайт «Мои соцсети»

Файлы находятся в этой папке. Описание:

- `index.html` — главная страница
- `styles.css` — стили
- `script.js` — логика показа интерактивных кнопок и видео

Как запустить локально:

1. Откройте `index.html` напрямую в браузере или запустите простой HTTP-сервер.
2. Для локальной проверки можно выполнить:

```powershell
python -m http.server 8000
```

Затем откройте: http://localhost:8000

Как опубликовать на GitHub Pages бесплатно:

1. Загрузите проект в GitHub в репозиторий.
2. В настройках репозитория откройте раздел Pages.
3. В Source выберите GitHub Actions.
4. После пуша в ветку `main` сайт будет опубликован автоматически через workflow из `.github/workflows/deploy-pages.yml`.
5. Ссылка будет выглядеть как `https://<ваш-логин>.github.io/<имя-репозитория>/`.

Что уже подготовлено для GitHub Pages:

- статичные HTML/CSS/JS без сборки и зависимостей;
- рабочие относительные пути к картинкам и видео;
- файл `.nojekyll` для исключения обработки Jekyll;
- workflow для автоматического деплоя в Pages.

Важно:
- Все медиа-файлы должны быть в папках `images/` и `video` внутри репозитория.
- Если у вас есть кастомный домен, добавьте файл `CNAME` в корень проекта.
- Для соцметок Open Graph замените `./` на реальный URL страницы после публикации, если хотите красивое превью в соцсетях.
